###  layers的概念
[第十二天: 创建你的第一个 Spacemacs Layer](http://book.emacs-china.org/#orgheadline84)
```
init-packages.el        # 插件管理
init-ui.el              # 视觉层配置
init-better-defaults.el # 增强内置功能
init-keybindings.el     # 快捷键绑定
init-org.el             # Org 模式相关的全部设定
custome.el              # 存放使用编辑器接口产生的配置信息
```
